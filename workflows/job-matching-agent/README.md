# 🧠 AI Resume-Based Job Matching Agent

This workflow analyzes a user's resume, finds relevant job opportunities, and sends curated results via email.

---

## 🔥 Features

* Extract skills from resume using AI
* Fetch jobs from API (Remotive)
* Match jobs based on skills
* Generate explanation for each job
* Send results via email

---

## 📂 Files

* `ai-job-matching-agent.json` → n8n workflow
* `workflow-preview.png` → workflow screenshot
* `sample-output.pdf` → example output

---

## 🚀 How It Works

1. User submits email + resume
2. AI extracts skills
3. Jobs are fetched & matched
4. AI explains job relevance
5. Email is sent with results

---

## 📩 Output

* Job title
* Company
* Apply link
* AI explanation

---

## 💡 Notes

* Designed for freshers
